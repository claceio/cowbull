{{ template "icon_cow" "w-6 h-6 text-primary" }} How to play

  • A hidden word has four letters, no repeats.
  • Enter guesses — valid English words, four letters, no repeated letters.
  • {{ template "icon_bull" "w-4 h-4" }} Bull — a letter is in the same position in your guess and the hidden word. Bulls-eye!
  • {{ template "icon_cow" "w-4 h-4" }} Cow — a letter is in the hidden word but at a different position.
  • Four bulls and the word is yours.

Example

If the hidden word is WORK:

  • ROSE → 1 bull (O), 1 cow (R)
  • CORD → 2 bulls (O, R), 0 cows
  • SEND → no match, four letters eliminated
  • WORK → 4 bulls, game over

Scoring

Every game starts at 10 points and loses points for the effort it took:

  • Each guess after the first costs 0.3.
  • Each hint costs 0.8 — use them as a last resort.
  • Time costs 0.05 per second (capped at 6 points), with a free 10-second allowance per guess. The clock starts at your first guess.
  • A win never scores below 1.

Try it

Playing with friends

  • Set up a game with 1–8 rounds and share the 4-letter code — everyone gets the same words in the same order.
  • Set your name (button at the top right) so friends can spot you on the board. It is saved on this device only — no account needed. Skip it and you get a random name like PAAEE.
  • Live standings and activity: you'll see when someone joins, gets 3+ bulls, or finishes.

Tournament

  • A single round is a quick match; more rounds make a tournament. Give the game a name when setting it up.
  • Your scores across rounds add up as a running tally — highest total wins.
  • Stopping early marks you DNF, but your total keeps counting: a strong early score can still win the tournament.
